The **energyBrain Dashboard** offers an intuitive way to visualize and control all entities and perform manual optimization simulations.

Below is a functional walkthrough.

### 1. Log in

Visit [**energy-app.erglocale.com**](http://energy-app.erglocale.com)\
Use your credentials to access your workspace.

### 2. Networks

- Go to **Networks → Create Network**
- Add details like grid capacity, solar capacity, inverter efficiency, and location coordinates.
- Once saved, view real‑time **Solar Forecast** for each site.\
  _(Screenshot: “Network map + solar forecast graph”)_

### 3. Charge Points

- Navigate to **Charge Points → Create Charge Point**
- Assign to an existing network.
- Enter max capacity, type (AC/DC), and connector details.
- All CSMS charge points requiring optimization should exist here.\

### 4. Vehicles

- Go to **Vehicles → Create Vehicle**
- Input VIN, license plate, battery size, target SoC, and max power.\
  _(Screenshot: Vehicle info card)_

> ⚠️ Vehicles must be onboarded for optimization algorithms to function.

### 5. Charging Sessions

- Go to **Sessions → Create New Session**
- Choose the vehicle and associated charge point connector.
- The session captures active charging data.\

### 6. Optimizations

- Go to **Optimizations → Select Network**
- View the **Solar Forecast** + **Predicted Power Usage**
- Adjust grid/solar limit overlays
- Click **View raw data** to get underlying schedule or export data via API

(Screenshot: forecast + cost graph)

> The graph combines Solar Power Forecast, Grid Prices, Power Usage, and the Optimized Schedule for that network.

### Example Dashboard Workflow

1. Create network
2. Add chargepoints → vehicles → sessions
3. Open **Optimizations tab**
4. View analytics and predicted schedules
5. Export data or compare it with real sessions
