##  **Overview**

The **energyBrain platform** by **ergLocale** allows smart energy forecasting and energy optimization for EV fleets and charging sites.\
You can interact with it using our **dashboard UI** or integrate our **REST API**.

This Getting Started guide walks you through:

- Prerequisites
- Authentication
- Sandbox (Try‑Before‑You‑Integrate) environment
- A simple step‑by‑step energy optimization use case using both UI and APIs.

### 1. Prerequisites

Before you begin, ensure you have:

- A registered **ergLocale Developer Account**
- Your **Client ID** and **Client Secret** credentials
- Access to either the **Production** or **Sandbox** API endpoints
- Basic knowledge of REST APIs and JSON requests
